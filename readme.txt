== Description ==

Plugin used to set urls and page and can viewable for a particular interval of time.

== Installation ==

1. Upload pageloader folder to your WP plugin folder `/wp-content/plugins/` directory
2. The path must look like this: `/wp-content/plugins/pageloader/`
3. Activate the plugin through the 'Plugins' menu in WordPress admin
4. You are ready, check the optins in option menu.
